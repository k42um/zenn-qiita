## はじめに
怠惰な筆者はゴロゴロしながら「今気温どれくらいなんだ...？」と思っても、温度計を見に行くのは面倒...みたいなタイミングがよくありました。（それくらいサボるなって話ではありますが...。笑）筆者の自宅は基本スマートホーム化を進めており、エアコンやテレビ、照明などは全てアレクサ経由で操作しています。ちなみにエアコンやテレビはもちろん古いもので、スマートフォンやスマートスピーカからの操作に対応していないので、後から購入した『Nature Remo Nano』を利用しています。これがとても便利で簡単にスマホやEchoから赤外線リモコンのようにエアコンなどを操作できます。一応リンクを貼っておきますので、よかったらご確認ください。（プライムセールなどでの購入がおすすめです！）
https://amzn.to/3ZbwtY7

いつものノリでAmazonでス⚪︎ッチボットなどを買おうとするとかなり高かった...。筆者は大学生なので気温測るためだけに1万円近くを払うことは厳しいため、家にあったラズパイとAmazonで買った気温・湿度センサを用いて、自作してみることにしました。

## 用意するもの
### Raspberry Pi
筆者はRaspberry Pi 3 Model Bを使用しました。インターネット接続できるものであれば、正直なんでも良いです。

### DHT22（温度・湿度センサ）
単体で購入しても良いのですが、抵抗等の準備が面倒だったのでAmazonでモジュールを購入しました。インターネットで調べた感じ、DHT22が一番ポピュラーと言っても問題なさそうです。

https://amzn.to/49kX76U

## 手順

:::message
ラズパイの初期設定はすでに完了しているものとします。「まだだよ...分かんないよ...。」という人はGoogleで『ラズパイ 初期設定』みたいに検索してください！本記事では省略いたします。
:::

### 0. ライブラリのインストール
以下のコマンドを使用して、必要なライブラリをダウンロードします。仮想環境を使用してpipのインストールを行うのが王道かつ正解ですが、面倒だったので`--break-system-packages`コマンドを使用しました。きちんとやりたい人は『ラズパイ pip 仮想環境』などでググってみてください。

```
pip install adafruit-circuitpython-dht --break-system-packages
```
