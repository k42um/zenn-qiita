## はじめに
Hugo（ヒューゴ）は世界一高速な静的サイトジェネレータです。と言っても良く分かりづらいので、分かりやすく言うと **「簡単にWebサイトを作成・保守するためのフレームワーク」** です。

![Hugoのホームページ](/images/how-to-use-hugo/hugo.png)
*Hugoの紹介（公式HPより引用）*

ブログやポートフォリオ、紹介ページといったWEBサイトを作るとなると、そこそこのWEBの関する知識（HTML, CSS, JavaScript, React, ...）が必要になるケースが多く、ハードルは高めです。最近は生成AIがものすごいスピードで進化しているので、頼めばかなり良いWEBサイトを作ってくれますが、その分メンテナンスが大変なイメージがあります。（生成AIの凄さに気を取られてしまい、よく忘れ去られるのがこの観点ですね。WEBサイトが大きくなり、そして長く続くにつれて、生成AIにメンテナンスさせるのが難しくなります。特に「全く分からんが、生成AIが上手いことやってくれてる」パターンだと、何か起こった際には対応できずにお蔵入りなんてケースも...。）

そこで、**Hugo**を使います。簡単に仕組みを話しておくと、まずHugoを用いてフレームワークを作成します。そして、その後数多くのテーマ（色合いやデザイン、サイトマップなど）の中から自分の気に入るものを探し、反映します。その後は**ただ追加したい記事をmarkdown形式で書いていくだけで、自動的にHTML等に変換され、世に発信**されます。自分みたいにとりあえずサクッとWEBページを作りたい時や、WEB知識のない団体のホームページを代々引き継いでいくパターンなどではかなり重宝します。そして、**無料**で使えるというのも嬉しいポイントの一つです。

筆者もHugoでサイトを作成するタイミングが何度かあったので、備忘録として本記事を書いております。メンテナンス性の高さとサイトのおしゃれさが両立でき、非常に満足しているのでぜひ**Hugo**を使ってみてください。

## 環境
筆者は`macOS Sequoia 15.7.4`でテストしました。

## やること
### Hugo のインストール
公式サイトのインストールガイドに沿って、操作を進めてください。
https://gohugo.io/installation/

MacOSの人はHomeBrew（パッケージマネージャ）を使って、
```bash
# HomeBrew がまだ入っていない人は先に実行
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Hugo のインストール
brew install hugo

# git のインストール
brew install git
```
と進めることでインストールできます。他のOSの人は上記ページを参考にやってみてください。

### サイトの作成
hugoのインストールを完了したら、特定のディレクトリで以下のコマンドを実行することでサイトのフレームワークを作ることができます。作成するサイト名はここでは`hugo-site`とします。

```bash
# 今後は hugo-site の部分を自身がつけたいサイト名に置き換えてください
hugo new site hugo-site
```

そうすることで、`hugo-site`ディレクトリ下にHugoのフレームワークが生成されます。そして、以下のコマンドを実行して、`hugo-site`ディレクトリに移動しましょう。

```bash
# hugo-site ディレクトリに移動
cd hugo-site
```

### テーマの選択
次に自身のWEBサイトに採用したいテーマを探します。以下のリンクからHugoのテーマの一覧が確認できるので、採用したいものを選びましょう。

https://themes.gohugo.io/

今回、筆者は「[Hugo Narrow](https://themes.gohugo.io/themes/hugo-narrow/)」というテーマを使用します。テーマによってダウンロード方法が異なることもあるので、詳細は選択したテーマのドキュメントを参照してください。

テーマを選択し、`Download`を選択すると、GitHubのページに飛びます。

![テーマの選択画面](/images/how-to-use-hugo/theme.png)
*テーマを選択されると、上記の画面が表示される*

その後、画面右上の`Code`を押し、`SSH`を押し、表示されるリンクをコピーしてください。

### テーマの設定
`hugo-site`のディレクトリで以下のコマンドを実行してください。

```bash
# themes ディレクトリへ移動する
cd themes

# submodule としてダウンロードする
git submodule add <コピーしたリンク>
```

もし、gitやGitHubの設定をしていないorしたくない場合はテーマをダウンロードしてきて、ディレクトリにほっぽりこんでも構いません。

以降の設定はテーマによって、大きく異なるため、省略します。GitHub Starsの多いテーマ（=人気のテーマ）は大体丁寧なドキュメントを有しているのでそれを参考にしましょう。時々日本語のドキュメントがついているテーマもあるので、ぜひ色々探してあげてください。

### サイトのテスト
テーマに沿ったサイトの設定やページの執筆を完了したのちは、

```bash
hugo server -D
```

というコマンドを実行したのちに、

http://localhost:1313

にアクセスすることで、実際にWEBサイトがどのように表示されるかを確認することができます。

### GitHub Pages へのデプロイ
詳細は省略しますが、HugoをGitHub上にアップし、GitHub Actionsを適切に設定することで、GitHub Pagesに自身のサイトをあげることができます。無料で公開できるので、ぜひ使ってみてください。

詳細は以下より、確認してください。
https://github.com/peaceiris/actions-hugo
